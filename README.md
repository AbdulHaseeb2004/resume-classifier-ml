# Resume Classification & Skill Matching System

This project was developed during my AI-ML internship at Hulk Hire Tech.  
It classifies resumes into job roles and extracts skills using NLP & ML.

## Features
- Converts PDF resumes to text
- Classifies resumes into roles (Data Scientist, ML Engineer, etc.)
- Extracts skills using Named Entity Recognition (NER)
- Web interface for easy upload and prediction

## Tech Stack
- Python, Flask, Streamlit
- Scikit-learn, spaCy, NLTK
- Pandas, NumPy
- GitHub, Vercel (for deployment)

## How to Run
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
